Getting Started

https://developers.google.com/appengine

  1. Download the SDK.
  2. Sign up for an account.
  3. Read the getting started information for your language.
  4. Check out the rest of the App Engine documentation.

Let's Show How in 5 minutes or less

  1. Open the Cloud Developers Console
    [ http://cloud.google.com/console ]
  2. Click Create Project
  3. Enter a project name and a project ID
  4. Click Create

Push To Deploy in 8 minutes or less

  1. Create Your Project on GitHub
  2. Install Git
  3. Open the Cloud Developers Console
    [ http://cloud.google.com/console ]
  4. Select Your Project
  5. Select Cloud Development > Releases in the left-hand navigation panel

  6. Click Connect a GitHub Repo.

  7. Enter the GitHub repository URL

  8. Click Connect.

  9. Authorize access
